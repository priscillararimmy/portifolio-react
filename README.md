## 🚀 Requisitos

- NodeJS

## Como rodar o projeto

No terminal, execute os comandos:

```bash
yarn i
yarn start
```

Ou usando o NPM:

```bash
npm i
npm start
```

## 🚀 Tecnologias

- HTML
